# Vue 특징

1. 컴포넌트 기반의 개발 방식
   - 화면을 여러 단위로 나누어 개발한다.
   - 재사용성, 구현 속도, 가독성의 향상
2. MVVM 방식
3. 앵귤러와 리액트 장점을 흡수
   - 가장 최근에 나온 라이브러리 (2014)
   - 2-way data binding
   - Virtual DOM (리액트)
4. Vue는 싱글 파일 컴포넌트이다.
   - template
   - script
   - style
     - scoped는 컴포넌트에만 적용되는 CSS

# React와 다른 점

1. React는 렌더링할 해당 상태 변수의 값을 바꾸더라도 렌더링되지 않는다.
   - useState와 같은 훅 함수를 이용하여 값을 설정해줘야만 렌더링을 실행한다.
